# alyson_xls_parser
Reads and Writes .xls files


Download the files:
Two ways to download
Using zip `Code > Download Zip`
Using git
```
git clone https://github.com/Palt0n/alyson_xls_parser.git
```

Install python libraries
```
pip install xlrd
pip install xlwt
```

```
pip install -r requirements.txt
```
